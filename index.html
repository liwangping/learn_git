Git is a distributed(分布式) version control system
Git is a free software

git init

初始化
建立一个空库

cd .git

进入git的隐藏库

cd ../

进入上层文件

git add

添加文件 此前都是缓存区中。相当于购物车

git commit -m "wrote a readme file"

确认添加文件 相当于付款确定

git status

可以确认下当前仓库的情况 可以看到有哪些文件出现了改变

git diff 文件名

用来对比上次提交的文件的具体差别。

git log

仓库日志 g 上 k 下 q 退出

git log --pretty=oneline

为对日志历史简单修饰

git reset --hard HEAD^

指向上个版本，一个^表示上一个 这个命令因为无法解析^所以使用gitbash

git reset --hard 前五位  版本号

指向想要的版本

git reflog

将所有版本号保存

git checkout --readme.txt

将其从缓冲区撤回

可以先git reset 再 checkout

git rm

删除文件

git remote add origin git@github.com:lwpwyf/learn_git.git

//类似于此与github仓库相连接

git push -u origin master

将其提交上去。

第二次提交到github时不再使用
untracked file 未在版本库里的文件
